# ICompras e-commerce
### Este projeto se trata de de um conjunto de microsserviços que compõem o sistema de e-commerce ICompras, que se comunicam de forma assíncrona por meio do Apache Kafka.

# Microsserviços
- [Clientes](https://github.com/Leoo098/icompras-clientes)

- [Produtos](https://github.com/Leoo098/icompras-produtos)

- [Pedidos](https://github.com/Leoo098/icompras-pedidos)

- [Logística](https://github.com/Leoo098/icompras-logistica)

- [Faturamento](https://github.com/Leoo098/icompras-faturamento)
